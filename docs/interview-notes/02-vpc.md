# AWS VPC Interview Notes

## What is a VPC?

A Virtual Private Cloud (VPC) is a logically isolated network inside AWS.

All AWS resources are deployed inside a VPC.

## Why Create a Custom VPC?

Production environments should never rely on the default VPC.

Benefits

- Better Security
- Network Isolation
- Full Control
- Scalability
